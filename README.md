# 网易云音乐爬虫

根据歌单url进行歌曲下载

## 使用方法

![Snipaste_2022-02-13_20-39-15](https://haochen1204.oss-cn-chengdu.aliyuncs.com/web_picture/img/Snipaste_2022-02-13_20-39-15.png)

如果需要爬取我喜欢的音乐，需创建一个新歌单，将其设置为公开，然后将我喜欢的音乐放入该歌单，再对该歌单进行爬取。
